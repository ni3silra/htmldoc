Simple Setup Example

The absolute minimum code needed to create a diagram
What you'll learn:
  • How to include the library with just one script tag
  • The minimal HTML needed for a working diagram
  • Basic initialization with default settings
  • How to create a simple 3-component architecture

Step 1: Include the Library

Add this single script tag to your HTML page (no installation required):

<script src="https://cdn.jsdelivr.net/npm/html-diagram-library@1.0.0/dist/html-diagram-library.min.js"></script>

Step 2: Create a Container

Add a div with width and height where your diagram will appear:

<div id="diagram" style="width: 800px; height: 500px;"> <!-- Your diagram elements will go here --> </div>

Step 3: Define Your Components

Use custom HTML tags to define your architecture components:

<div id="diagram" style="width: 800px; height: 500px;"> <api-gateway name="Main Gateway" brand="kong" connects="web-service" id="gateway"> </api-gateway> <microservice name="Web Service" brand="spring-boot" connects="database" id="web-service"> </microservice> <database name="Main Database" brand="postgresql" id="database"> </database> </div>

Step 4: Initialize and Render

Add a script to create and render the diagram:

<script> // Create diagram with default settings const diagram = new DiagramLibrary({ container: '#diagram' }); // Render the diagram diagram.render(); </script>

Complete Copy-Paste Code

Here's the complete HTML file you can copy and save as a .html file:

<!DOCTYPE html> <html lang="en"> <head> <meta charset="UTF-8"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> <title>My First Diagram</title> </head> <body> <h1>My Architecture Diagram</h1> <div id="diagram" style="width: 800px; height: 500px; border: 1px solid #ccc;"> <api-gateway name="Main Gateway" brand="kong" connects="web-service" id="gateway"> </api-gateway> <microservice name="Web Service" brand="spring-boot" connects="database" id="web-service"> </microservice> <database name="Main Database" brand="postgresql" id="database"> </database> </div> <script src="https://cdn.jsdelivr.net/npm/html-diagram-library@1.0.0/dist/html-diagram-library.min.js"></script> <script> const diagram = new DiagramLibrary({ container: '#diagram' }); diagram.render(); </script> </body> </html>
